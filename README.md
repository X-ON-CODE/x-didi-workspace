# 🎯 个人工作管理系统

> 基于 DDO 目标驱动的个人工作管理仓库

## 简介

这是一个 All-in-One 的个人工作管理方案，帮助你：
- 记录每日工作（日记）
- 管理项目和待办
- 跟踪 DDO 目标进度
- 自动生成周期性汇报

## 快速开始

### 1. 配置 DDO（首次使用）

编辑 `ddo/current.md`，定义你的目标（类似 OKR）。

### 2. 日常使用

```bash
# 告诉 AI：
"记录今天完成了 xxx"
"查看我的待办"
"生成本周周报"
"分析 DDO 偏差"
```

### 3. 查看汇报

生成的周报、季度汇报、年度总结分别存放在：
- `weekly/` - 周报
- `quarterly/` - 季度汇报
- `yearly/` - 年中/年终总结

## 目录说明

| 目录 | 用途 |
|------|------|
| `ddo/` | DDO 目标定义与追踪 |
| `daily/` | 每日工作日记 |
| `projects/` | 项目文档 |
| `todos/` | 待办事项 |
| `weekly/` | 周报输出 |
| `quarterly/` | 季度汇报输出 |
| `yearly/` | 年中/年终总结输出 |
| `templates/` | 各类文档模板 |

## 工作流程

```
定义 DDO ──→ 日常记录 ──→ 项目/待办管理 ──→ 定期生成汇报
    ↑___________________________________________|
         （通过偏差分析持续优化方向）
```
