## 🔄 核心工作流程

### 1. 日常工作流程

**操作指令**：`记录日记` / `写日报`

**AI 执行步骤**：
1. 检查 `ddo/current.md` 了解当前目标权重
2. 检查 `todos/current.md` 了解当前待办池状态
3. **记录当天实际完成的内容**到日记文件 `daily/YYYY/MM-DD.md`
4. 更新 `todos/current.md`：将已完成的任务移到「已完成」，更新进行中的任务状态
5. 关联相关项目和 DDO
6. 如有唐欢相关工作，单独标注

### 2. 周报生成流程

**周四汇报（业务+技术）**
- **指令**：`生成周四周报` / `写业务技术周报`
- **统计周期**：上周五 至 本周四
- **模板**：`templates/weekly-thursday.md`
- **输出**：`weekly/YYYY-WXX-thursday.md`
- **包含**：
  1. 业务汇报（需求列表格式）
  2. 技术汇报（里程碑格式）
- **⚠️ 重要**：周四必须在 **前端周会（19:00）之前** 完成周报，建议安排在 **17:00-18:00**

**周五PMO表（单独）**
- **指令**：`生成PMO表` / `写资源投入表`
- **统计周期**：本周一 至 本周五
- **模板**：`templates/weekly-pmo.md`
- **输出**：`weekly/YYYY-WXX-pmo.md`
- **包含**：人周投入统计表（总和=1）
- **⚠️ 重要**：每周五必须在 **18:00下班前** 完成并提交

**PMO填写规则**：
- 一般需求使用具体需求名称
- 规划里找不到需求名称的，归类到以下任务：
  - **费控客户关系维护**：费控相关的线上问题
  - **【产研】产品售前&预研支持**：不好分类的调研性工作
  - **集成平台技术任务**：[T-QYJ-575980](https://ddp.intra.xiaojukeji.com/issue/story/T-QYJ-575980)
  - **开放平台技术任务**：[T-QYJ-575984](https://ddp.intra.xiaojukeji.com/issue/story/T-QYJ-575984)

### 3. 季度汇报生成流程

**操作指令**：`生成本季度汇报` / `写季度总结`

**AI 执行步骤**：
1. 读取 `templates/quarterly.md` 模板
2. 按三部分整理：
   - **Q1工作回顾**：总结性呈现
   - **反思与总结**：Highlight（做得好的）、Lowlight（做得不够好的）、Detail（日常产出）
   - **Q2规划**：下季度规划
3. Detail部分围绕：
   - 日常开发
   - 技术产出
   - 产出质量（可选）
   - 团队贡献（可选，非开发工作类产出）
4. 输出到 `quarterly/YYYY-QX.md`

### 4. 待办池管理流程

**操作指令**：`更新待办` / `查看待办池`

**AI 执行步骤**：
1. 读取 `todos/current.md`
2. 添加新任务到「待办」列表
3. 更新进行中任务的状态
4. 将已完成的任务移到「已完成」列表，记录完成日期
5. 更新风险和里程碑状态

### 5. DDO 目标管理流程

**操作指令**：`更新 DDO` / `查看目标进度`

**AI 执行步骤**：
1. 读取 `ddo/current.md`
2. 根据当前工作进展更新各指标状态
3. 计算目标达成率
4. 分析偏差原因

### 6. 项目管理流程

**操作指令**：`创建项目 [项目名]` / `更新项目 [项目名]`

**AI 执行步骤**：
1. 根据项目所属业务（费控/集成平台/开放平台）选择目录
2. 创建/更新项目文档
3. 关联相关 DDO 指标
4. 更新里程碑和进度

### 7. 唐欢工作管理流程

**操作指令**：`查看唐欢工作安排` / `安排唐欢任务` / `唐欢本周产出`

**AI 执行步骤**：
1. 检查唐欢当前参与的项目（集成平台、开放平台）
2. 根据项目进度和她的能力特点分配任务
3. 记录她的产出和成长
4. 给出培养建议
